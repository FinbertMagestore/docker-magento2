# Docker for Magento
Docker image for Magento.

## Use batch script
### Build images
```bash
./bin/build.sh
```

### Run service
```bash
./bin/run.sh
```

### Install magento 2
```bash
./bin/ssh.sh
./install_magento.sh
```

### Stop service
```bash
./bin/stop.sh
```

### Remove service
```bash
./bin/remove.sh
```

### SSH
```bash
./bin/ssh.sh
```
